# QQ经典农场助手 - UI/UX 规范

## 1. 整体布局

单窗口应用，固定尺寸 1000x700，不可调整大小。采用左侧导航 + 右侧内容区的经典布局。

```
┌──────────────────────────────────────────────────┐
│  QQ经典农场助手          ─  □  ×  │
├────────┬─────────────────────────────────────────┤
│        │                                         │
│  导航栏 │              内容区                     │
│        │                                         │
│ 🏠 首页 │                                         │
│ ⚙ 设置 │                                         │
│ 📋 日志 │                                         │
│        │                                         │
│        │                                         │
│────────│                                         │
│ 状态指示 │                                         │
│ ● 在线  │                                         │
└────────┴─────────────────────────────────────────┘
```

### 1.1 左侧导航栏（宽度 72px）

- 图标式导航，鼠标悬停显示文字提示
- 三个页面：首页、设置、日志
- 底部显示连接状态指示灯（绿色在线/红色离线/黄色重连中）

### 1.2 窗口控制

- 自定义标题栏（无边框窗口）
- 关闭按钮 → 最小化到系统托盘（不退出）
- 托盘右键菜单：显示窗口 / 退出

- 功能开关：Switch 组件，切换后立即生效
- 种植策略：切换「自动最优」时，系统自动计算并显示推荐作物；切换「手动选择」时，下拉框列出当前等级可购买的所有作物
- 最近操作：只显示最新 5 条，点击「查看全部」跳转到日志页

## 3. 视觉规范

### 3.1 配色方案

采用深色主题，护眼且适合长时间运行：

| 用途 | 颜色 |
|------|------|
| 背景色 | #1a1a2e |
| 侧边栏背景 | #16213e |
| 卡片背景 | #0f3460 |
| 主色调（强调） | #e94560 |
| 文字主色 | #eee |
| 文字次色 | #a0a0a0 |
| 成功/在线 | #67c23a |
| 警告 | #e6a23c |
| 错误/离线 | #f56c6c |

### 3.2 字体

- 主字体：系统默认（Microsoft YaHei）
- 日志区域：等宽字体（Consolas / monospace）
- 字号：正文 14px，标题 16px，日志 13px

### 3.3 组件状态

| 组件 | 默认 | 悬停 | 禁用 |
|------|------|------|------|
| Switch 开关 | 灰色底 | 轻微高亮 | 半透明 |
| 按钮 | 主色调底 | 加亮 | 灰色底 |
| 输入框 | 深色底+边框 | 边框高亮 | 灰色底 |

## 4. 用户流程

### 4.1 首次使用流程

```
启动应用 → 显示首页（未登录状态）
  → 输入 code + 选择平台 → 点击连接
  → 连接成功 → 状态卡片更新 → 功能自动开始运行
  → 最小化到托盘 → 后台持续运行
```

### 4.2 重连流程

```
连接断开 → 状态指示灯变红 → 登录区重新展开
  → 用户输入新 code → 点击连接 → 恢复运行
```

### 4.3 日常使用流程

```
双击托盘图标 → 显示窗口 → 查看状态/日志 → 关闭窗口（回到托盘）
```
